# Photos Directory

Add your vehicle photos here with these exact filenames:

## Required Files:

1. **slate-executive-s.jpg** - Slate Executive S vehicle image
2. **atlas-rover-x.jpg** - Atlas Rover X vehicle image
3. **phantom-s-class.jpg** - Phantom S Class vehicle image
4. **horizon-transit.jpg** - Horizon Transit vehicle image

## Image Specifications:

- **Aspect Ratio**: 16:10 (recommended for proper display)
- **Format**: JPG or PNG
- **Minimum Width**: 600px
- **Recommended Width**: 800px - 1200px
- **Quality**: High quality (optimized for web)

## How It Works:

1. Place your vehicle photos in this folder with the exact filenames listed above
2. The `data/vehicles.json` file references these photos
3. The vehicles.html page automatically fetches and displays them

## Example:
```
photos/
├── slate-executive-s.jpg
├── atlas-rover-x.jpg
├── phantom-s-class.jpg
└── horizon-transit.jpg
```

## Notes:

- If a photo is missing, it will show a broken image placeholder
- You can update photos anytime by replacing the files
- Changes will be reflected immediately in the website
- Make sure filenames match exactly (case-sensitive on Linux/Mac)
